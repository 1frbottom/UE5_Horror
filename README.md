    -> 게임 [ 타이틀 - 인게임(일시정지) - 종료(사망 or 클리어) ]<br>
    -> 기본 조작 [ 걷기, 달리기(스태미나), 숙이기 ]<br>
    -> 상호 작용 [ 아이템, 서랍, 문, 트리거 ]<br>
    -> 고스트 [ 탐색, 인지(sight, hearing), 공격 ]
	
- 조작방법<br>
	-> WASD : 이동, LCtrl : 숙이기, LShift : 달리기<br>
	-> F : 상호작용, ESC : 일시정지
